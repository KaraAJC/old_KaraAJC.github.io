    Pulling this site up by it's BOOTSTRAPS

    Projects GitItGurl, NerdLife, Bootstrap
    0 Comments December 8, 2014 6:05 PM

    Building a website is HARD work! But it's also very rewarding..

    I started my last week knowing that putting all the feedback I've recieved into my site was going to be quite the feat. Having a visual site was very important to me, and my colleagues/friends/etc. so I started there. Then, I looked at how Bootstrap works, and watched a few videos on how to get started. Upon further research I found a lot of great resources to understand how customizeable Bootstrap is. I used a tool called Bootply to modify my HTML and CSS together with a live preview to understand what I was doing.

    It was truly nerve-wrecking to make this drastic a change to my site repo. I was really worried it was going to loose all the work I had done. What I did in addition to making a copy of my original site, was to nest the old site in a folder, so that I can still refer to the blogposts as I update my site.

    My next steps will be to understand how Jekyll works and utilize it's paging features to continue to keep my blog updated.
